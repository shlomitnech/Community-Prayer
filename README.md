# Community-Prayer
Link to website (In progress): https://dev.dib3ff9kiwm7v.amplifyapp.com
About: I am creating forum online using AWS where people can submit names that need to be prayers for various reasons (finding a spouse, having children, sickness and financial problem)
The data is collected using AWS amplify, then through lambda functions sent to the dynamodb table
I also have a lambda function that when complete will send out a weekly email to everyone subscribed the names of the people to pray for for the week. Every week the subject will rotate
I have only just begun the project so there are many parts that I wish to change and make better.
